# 테스트의 목적


- 인수 테스트
    - 클라이언트 요청에 정상적으로 응답하는지 확인한다.
    - API 명세서 역할을 대신할 수 있도록 깔끔하게 작성한다.
- 서비스 테스트
    - 어떠한 협력이 이루어지는지 확인한다
    - 협력 객체 간 어떤 메시지를 통해 비즈니스 로직이 제대로 동작하는지 확인한다.
- 도메인 테스트
    - 객체 상태에 따른 비즈니스 로직이 제대로 동작하는지 확인한다.
- 레포지토리 테스트
    - 쿼리가 정상 동작하는지 확인한다.
